# Mental Math 🧠

Train your brain with this simple and fast mental math trainer.

Mental Math is a Progressive Web App (PWA) designed to help you improve your mental arithmetic skills. It covers basic operations: addition, subtraction, multiplication, and division.

## Features

- **Four Operations**: Practice addition, subtraction, multiplication, and division.
- **Dynamic Difficulty**: Problems are generated with random numbers within a reasonable range for mental training.
- **Instant Feedback**: Know immediately if your answer is correct.
- **PWA Ready**: Can be installed on your home screen for offline use and quick access.
- **Glassmorphism Design**: A modern, sleek interface with smooth transitions.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling with custom properties and gradients.
- **JavaScript**: Vanilla JS for logic and interactivity.
- **Web App Manifest**: For PWA capabilities.

## How to Run

Simply open `index.html` in any modern web browser.
